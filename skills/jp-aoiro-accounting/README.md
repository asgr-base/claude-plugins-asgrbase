# jp-aoiro-accounting

日本の個人事業主・フリーランス向け青色申告帳簿作成ツール。
Markdownテーブルで仕訳帳を管理し、Pythonスクリプトで帳簿・決算書を自動生成する。

## なぜこのツール？

従来の会計ソフト（クラウド型・デスクトップ型）には以下の課題がある:

- **ベンダーロックイン**: データが専用形式で閉じ込められ、エクスポートに制約がある
- **AIエージェントとの連携困難**: Web UIベースのため、AIが直接データを操作できない
- **ランニングコスト**: 無料プランには制限があり、継続利用には月額費用が発生する

**jp-aoiro-accounting** はこれらを解決する:

| 特徴 | 従来の会計ソフト | 本ツール |
|------|----------------|---------|
| データ形式 | 専用フォーマット | **Markdown**（テキストエディタで編集可能） |
| AI連携 | Web UI操作のみ | **Claude Code Skill**としてAIが直接記帳・検証・決算 |
| コスト | 月額1,000〜3,000円程度 | **無料**（MIT License） |
| 依存関係 | ブラウザ＋インターネット接続 | **Python標準ライブラリのみ**（オフラインで動作） |
| バージョン管理 | 不可 or 限定的 | **Git管理可能**（差分追跡・履歴管理） |
| カスタマイズ | 限定的 | **科目マスタで自由に追加・変更** |

## 特徴

- **Markdownベース**: 仕訳帳・台帳をMarkdownテーブルで管理（Obsidian、VS Code等で閲覧・編集可能）
- **AIエージェント連携**: Claude Code Skillとして、AIが記帳・検証・決算処理を支援
- **依存ライブラリなし**: Python標準ライブラリのみで動作（pip install不要）
- **青色申告対応**: 最大65万円控除（令和9年分以降は最大75万円）に必要な複式簿記帳簿（総勘定元帳・損益計算書・貸借対照表）を生成
- **カスタマイズ可能**: 勘定科目は科目マスタで自由に追加・変更
- **Git管理対応**: すべてのデータがプレーンテキストのため、変更履歴の追跡が可能
- **オフライン動作**: インターネット接続不要、ローカル環境で完結

## 前提条件

- Python 3.8以上
- テキストエディタ（Obsidian、Cursor、VS Code等）
- Claude Code（AIエージェント連携時）

## クイックスタート

### 1. 初期化

```bash
python3 scripts/aoiro.py init --year 2025 --output-dir ~/accounting/2025
```

以下のファイルが作成されます:
- `仕訳帳.md` - 仕訳入力用テーブル
- `科目マスタ.md` - 勘定科目定義
- `固定資産台帳.md` - 減価償却対象資産
- `家事按分設定.md` - 家事按分率

### 2. 仕訳入力

`仕訳帳.md` のテーブルに行を追加:

```markdown
| 日付 | 借方科目 | 借方金額 | 貸方科目 | 貸方金額 | 摘要 |
|------|---------|---------|---------|---------|------|
| 2025-09-30 | 売掛金 | 500,000 | 売上高 | 500,000 | A社 9月分報酬 |
| 2025-10-31 | 普通預金 | 448,950 | 売掛金 | 448,950 | A社 9月分入金 |
| 2025-10-31 | 事業主貸 | 51,050 | 売掛金 | 51,050 | A社 源泉徴収税 |
```

### 3. バリデーション

```bash
python3 scripts/aoiro.py validate ~/accounting/2025/仕訳帳.md
```

### 4. 帳簿生成

```bash
python3 scripts/aoiro.py generate ~/accounting/2025/仕訳帳.md --output-dir ~/accounting/2025/output
```

### 5. 決算書生成

```bash
python3 scripts/aoiro.py settlement ~/accounting/2025/仕訳帳.md --output-dir ~/accounting/2025/output
```

## 年次決算フロー

1. 通常仕訳の入力完了
2. `validate` でエラーチェック
3. `depreciation` で減価償却仕訳を生成 → 仕訳帳に追記
4. `allocation` で家事按分仕訳を生成 → 仕訳帳に追記
5. `validate` で再検証
6. `generate` で総勘定元帳・残高試算表を生成
7. `settlement` で損益計算書・貸借対照表を生成
8. 決算書の数値をe-Taxに手入力

## Claude Code Skill として使用

`.claude/skills/` にシンボリックリンクを配置:

```bash
ln -s /path/to/jp-aoiro-accounting ~/.claude/skills/jp-aoiro-accounting
```

これにより、Claude Codeが以下を支援:
- 仕訳帳への記帳（Markdownテーブル編集）
- バリデーション実行
- 帳簿・決算書の自動生成
- 仕訳パターンの提案

## 法令準拠

本ツールは以下の法令に基づいて処理を行います。詳細は [REFERENCE.md](REFERENCE.md) の「適用法令・税制改正情報」を参照してください。

| 処理 | 根拠法令 |
|------|---------|
| 帳簿形式（複式簿記） | 所得税法 第148条、第149条 |
| 青色申告特別控除 | 租税特別措置法 第25条の2 |
| 減価償却（定額法） | 所得税法施行令 第129条 |
| 非業務用資産の転用 | 所得税法施行令 第135条 |
| 耐用年数・償却率 | 減価償却資産の耐用年数等に関する省令 |

**税制改正の影響**:
- 令和9年分（2027年分）以降、青色申告特別控除が最大75万円に改正予定（優良な電子帳簿保存が要件）
- 税制改正により控除額・計算方法が変わる場合があります。確定申告前に最新の法令を確認してください

## 注意事項

- 本ツールは帳簿作成の補助ツールです。税務申告の正確性は利用者の責任です
- 消費税（インボイス制度）には非対応です（免税事業者を想定）。課税事業者の場合は別途消費税処理が必要です
- 不明な税務処理は税理士に相談してください
- e-Taxへの電子申告は手入力です（API連携なし）
- 電子帳簿保存法の「優良な電子帳簿」要件を直接満たすものではありません
