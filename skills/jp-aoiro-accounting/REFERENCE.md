# リファレンス: 勘定科目体系・仕訳パターン集

## 適用法令・税制改正情報

本ツールの帳簿処理は以下の法令に準拠する。税制改正により要件・控除額が変更される場合があるため、確定申告前に最新情報を確認すること。

### 根拠法令

| 法令 | 条文 | 内容 |
|------|------|------|
| 所得税法 第143条 | 青色申告 | 青色申告の承認に関する規定 |
| 所得税法 第148条 | 帳簿書類 | 青色申告者の帳簿書類の備付け・保存義務 |
| 所得税法 第149条 | 決算書添付 | 貸借対照表・損益計算書等の添付義務 |
| 租税特別措置法 第25条の2 | 青色申告特別控除 | 控除額・適用要件の規定 |
| 所得税法施行令 第129条 | 定額法 | 定額法による減価償却費の計算方法 |
| 所得税法施行令 第130条 | 定率法 | 定率法による減価償却費の計算方法 |
| 所得税法施行令 第135条 | 非業務用資産転用 | 業務の用に供した資産の償却費の特例 |
| 減価償却資産の耐用年数等に関する省令 | 別表 | 法定耐用年数・償却率 |
| 電子帳簿保存法（電子計算機を使用して作成する国税関係帳簿書類の保存方法等の特例に関する法律） | - | 電子帳簿の保存要件 |

### 青色申告特別控除（租税特別措置法 第25条の2）

控除額は記帳方式・申告方法により異なる。

**令和7年分〜令和8年分（2025〜2026年分）**:

| 控除額 | 要件 |
|--------|------|
| 65万円 | 正規の簿記（複式簿記）+ e-Tax申告 または 優良な電子帳簿保存 |
| 55万円 | 正規の簿記（複式簿記）+ 紙で申告 |
| 10万円 | 簡易簿記 |

**令和9年分以降（2027年分〜）** ※令和8年度税制改正大綱:

| 控除額 | 要件 |
|--------|------|
| 75万円 | 正規の簿記（複式簿記）+ e-Tax申告 + 優良な電子帳簿保存 |
| 65万円 | 正規の簿記（複式簿記）+ e-Tax申告 |
| 10万円 | 簡易簿記（55万円控除は廃止） |

**優良な電子帳簿の要件**: (1) 訂正等の履歴が残ること (2) 帳簿間で相互関連性があること (3) 日付・金額・相手方による検索機能があること

**本ツールの対応**: 本ツールで作成する帳簿は正規の簿記（複式簿記）に対応。e-Tax申告と組み合わせることで65万円控除の要件を満たす。75万円控除には別途、優良な電子帳簿保存法対応の電子帳簿保存が必要。

### 基礎控除の改正（令和7年分〜）

令和7年度税制改正により、基礎控除額が引き上げられた（所得税法 第86条）。

| 合計所得金額 | 改正後（令和7年分） | 改正前 |
|-------------|-------------------|--------|
| 132万円以下 | 95万円 | 48万円 |
| 132万円超〜336万円以下 | 88万円 | 48万円 |
| 336万円超〜489万円以下 | 68万円 | 48万円 |
| 489万円超〜2,400万円以下 | 58万円 | 48万円 |
| 2,400万円超〜2,450万円以下 | 38万円 | 32万円 |
| 2,500万円超 | 0円 | 0円 |

※令和9年分以降は合計所得金額2,350万円以下の場合一律58万円に収束予定。

### 電子帳簿保存法（2024年1月〜完全義務化）

電子取引（メール・Web等で受領した請求書・領収書等）のデータは、電子データのまま保存することが義務化された。個人事業主も対象。

- **対象**: 電子取引で授受した国税関係書類（請求書、領収書等）
- **保存要件**: 真実性の確保（タイムスタンプ等）+ 可視性の確保（検索機能等）
- **検索要件の緩和**: 基準期間の売上高5,000万円以下の場合、検索要件は不要

### インボイス制度（2023年10月〜）

本ツールは消費税（インボイス）に非対応であり、免税事業者を主な利用者として想定している。課税事業者（適格請求書発行事業者）の場合は別途消費税の処理が必要。

- **2割特例**: 免税事業者から課税事業者になった場合、納税額を売上税額の2割にできる特例（令和8年9月30日を含む課税期間まで）
- **3割特例**: 令和8年度税制改正大綱により、個人事業者については3割に引き上げた上で令和10年度（2028年度）まで延長予定

### 元入金の計算

期首の元入金は前年末から以下で計算:

```
今期の元入金 = 前期末の元入金
             + 前期の青色申告特別控除前の所得金額
             + 前期末の事業主借
             - 前期末の事業主貸
```

開業初年度は `元入金 = 事業開始時の資産合計 - 負債合計`

## 減価償却の計算方法

**根拠法令**: 所得税法施行令 第129条（定額法）、第130条（定率法）、第135条（非業務用資産転用時の特例）

### 定額法（個人事業主のデフォルト）

個人事業主は届出がない場合、定額法が法定償却方法（所得税法施行令 第120条の2）。

```
年間償却額 = 取得価額 × 定額法償却率
月割り額 = 年間償却額 × 事業月数 / 12
事業供用分 = 月割り額 × 事業専用割合
```

※平成19年4月1日以後取得の資産に適用（平成19年3月31日以前は旧定額法）。
※備忘価額（1円）まで償却可能。

### 非業務用資産の転用

個人使用していた資産を事業に転用する場合（所得税法施行令 第135条）:

```
転用時の未償却残高 = 取得価額 - (取得価額 × 0.9 × 旧定額法償却率 × 非業務用年数)
```

転用後は**取得価額を基礎**とし、定額法償却率で償却する。ただし償却累計が転用時の未償却残高を超えないよう制限する。

### 少額減価償却資産の特例

| 取得価額 | 処理方法 | 根拠 |
|---------|---------|------|
| 10万円未満 | 全額経費（消耗品費等） | 所得税法施行令 第138条 |
| 10万円以上20万円未満 | 一括償却資産（3年均等） | 所得税法施行令 第139条 |
| 30万円未満 | 少額減価償却資産の特例（青色申告者、年間合計300万円まで） | 租税特別措置法 第28条の2 |
| 30万円以上 | 通常の減価償却 | 所得税法施行令 第129条等 |

### 耐用年数と償却率（主な資産）

**根拠**: 減価償却資産の耐用年数等に関する省令（別表第一〜第八）

| 資産 | 耐用年数 | 定額法償却率 | 省令別表 |
|------|---------|------------|---------|
| 木造建物（住宅） | 22年 | 0.046 | 別表第一 |
| 鉄筋コンクリート建物 | 47年 | 0.022 | 別表第一 |
| パソコン | 4年 | 0.250 | 別表第一 |
| 事務机・椅子 | 15年 | 0.067 | 別表第一 |
| エアコン | 6年 | 0.167 | 別表第一 |
| 自動車（普通車） | 6年 | 0.167 | 別表第一 |

※償却率は平成24年4月1日以後取得の資産に適用される定額法償却率（1÷耐用年数、小数点以下第4位切上げ）。
