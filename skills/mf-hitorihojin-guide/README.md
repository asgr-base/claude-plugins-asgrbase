# mf-hitorihojin-guide

マネーフォワード クラウド ひとり法人プランの利用ガイド。

## What It Does

マネーフォワード クラウドの「ひとり法人プラン」に含まれるサービスの使い方、設定手順、機能制限を案内します。会計、給与、年末調整、社会保険など12サービスをカバー。

## Key Topics

- **プラン概要** — 年額払い・月額払い料金、含まれる12サービス
- **機能制限** — 年間仕訳数、AI-OCR回数、部門数、消費税申告
- **クイックスタート** — 会計初期設定、給与計算、年末調整、社会保険
- **FAQ** — 仕訳上限の上位プラン移行、消費税申告、電話サポート

## Usage

```
/mf-hitorihojin-guide
```

マネーフォワード、MFクラウド関連の質問時に使用。

## File Structure

```
mf-hitorihojin-guide/
├── SKILL.md          # Main guide
└── REFERENCE.md      # Detailed procedures and support URLs
```

## Requirements

- Claude Code CLI
- WebSearch tool（最新情報の取得に使用）
