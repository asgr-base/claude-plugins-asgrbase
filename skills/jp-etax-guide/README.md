# jp-etax-guide

e-Tax（国税電子申告・納税システム）の利用ガイド。

## What It Does

e-Taxを使った国税の電子申告・電子納付の手続きを案内します。確定申告、法人税、源泉所得税、消費税の申告から納付方法までカバー。Windows/macOS両環境に対応。

## Key Topics

- **ソフトウェア選択** — 確定申告書等作成コーナー / e-Taxソフト（WEB版）/ e-Taxソフト（インストール版）
- **環境別対応表** — Windows / macOS の機能差
- **macOSセットアップ** — マイナンバーカード、ICカードリーダー、JPKIクライアント
- **電子納付** — ダイレクト納付、Pay-easy、クレジットカード、スマホ決済
- **FAQ** — e-Tax vs eLTAXの違い、利用可能時間、ID/パスワード方式

## Usage

```
/jp-etax-guide
```

e-Tax、確定申告、国税関連の質問時に使用。
