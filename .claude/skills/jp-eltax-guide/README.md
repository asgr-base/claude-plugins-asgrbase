# jp-eltax-guide

eLTAX（地方税ポータルシステム）の利用ガイド。

## What It Does

eLTAXを使った地方税の電子申告・電子納付の手続きを案内します。給与支払報告書、法人住民税の申告、電子納付方法などをカバー。Windows/macOS両環境に対応。

## Key Topics

- **環境別対応表** — Windows DL版 / WEB版 / macOS WEB版の機能差
- **PCdesk設定** — Windows DL版のインストール・設定手順
- **macOS Safari設定** — ポップアップ・JavaScript・Cookie設定
- **電子納付** — インターネットバンキング、クレジットカード、ダイレクト納付
- **FAQ** — プラットフォーム選択、稼働時間、電子証明書

## Usage

```
/jp-eltax-guide
```

eLTAX、地方税、PCdesk関連の質問時に使用。

## File Structure

```
jp-eltax-guide/
├── SKILL.md          # Main guide
└── REFERENCE.md      # Detailed environment-specific settings
```

## Requirements

- Claude Code CLI
- WebSearch tool（最新情報の取得に使用）
