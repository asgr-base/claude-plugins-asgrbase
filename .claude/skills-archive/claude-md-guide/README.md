# claude-md-guide

Guide for creating and optimizing CLAUDE.md files to maximize Claude Code's effectiveness.

## What It Does

Provides best practices for writing CLAUDE.md — the project-level instruction file that Claude Code auto-loads in every conversation. Covers structure, placement, content strategy, and iterative improvement.

## Key Topics

- **60-line principle** — Optimal file size guidance
- **WHY/WHAT/HOW structure** — Recommended organization pattern
- **Progressive Disclosure** — Detailed info in separate linked files
- **Placement hierarchy** — `./CLAUDE.md`, `~/.claude/CLAUDE.md`, `.claude/rules/`
- **Include/exclude guidelines** — What belongs and what doesn't
- **Iterative improvement** — Workflow for continuous refinement

## Usage

```
/claude-md-guide
```

Use when creating a new CLAUDE.md or optimizing an existing one.

## File Structure

```
claude-md-guide/
├── SKILL.md          # Main guide
├── TEMPLATES.md      # Template collection
├── EXAMPLES.md       # Good/bad comparisons
└── CHECKLIST.md      # Creation/review checklist
```
