# aoiro.py スクリプト仕様

## 概要

Markdownテーブル形式の仕訳帳から帳簿・決算書を自動生成するCLIツール。
Python 3.8以上、標準ライブラリのみで動作。

## サブコマンド一覧

### validate

仕訳帳のバリデーション。

```bash
python3 aoiro.py validate <仕訳帳.md>
```

**チェック項目**:
- 各行の貸借一致（借方金額 = 貸方金額）
- 日付の形式（YYYY-MM-DD）
- 科目マスタに存在する科目か
- 金額ゼロの仕訳
- 摘要の空欄
- 期間外の日付

**終了コード**: 0 = OK, 1 = エラーあり

### generate

総勘定元帳・残高試算表を生成。

```bash
python3 aoiro.py generate <仕訳帳.md> --output-dir <出力先>
```

**出力ファイル**:
- `総勘定元帳.md` - 科目ごとの取引一覧と残高
- `残高試算表.md` - 全科目の借方・貸方合計と残高

### settlement

損益計算書・貸借対照表を生成。

```bash
python3 aoiro.py settlement <仕訳帳.md> --output-dir <出力先>
```

**出力ファイル**:
- `損益計算書.md` - 収入・経費・所得金額
- `貸借対照表.md` - 資産・負債・資本の期末残高、貸借検証

### depreciation

固定資産台帳から減価償却費を計算し、決算仕訳を出力。

```bash
python3 aoiro.py depreciation <固定資産台帳.md> --year <年>
```

**対応**:
- 定額法・定率法
- 月割り計算
- 非業務用資産の転用（転用時残高ベース）
- 事業専用割合

**出力**: stdoutに決算仕訳のMarkdownテーブルを出力。仕訳帳にコピー&ペーストで追記。

### allocation

家事按分設定に基づき按分仕訳を生成。

```bash
python3 aoiro.py allocation <仕訳帳.md> --config <家事按分設定.md>
```

**動作**: 各按分対象科目の年間合計に対し、私用分を `事業主貸` に振り替える仕訳を生成。

**出力**: stdoutに決算仕訳のMarkdownテーブルを出力。

### init

テンプレートから新年度の会計データフォルダを作成。

```bash
python3 aoiro.py init --year <年> --output-dir <出力先>
```

**作成されるファイル**:
- `仕訳帳.md`
- `科目マスタ.md`
- `固定資産台帳.md`
- `家事按分設定.md`
- `output/`（帳簿出力先ディレクトリ）

## Markdownテーブルの形式

### 仕訳帳

```markdown
| 日付 | 借方科目 | 借方金額 | 貸方科目 | 貸方金額 | 摘要 |
|------|---------|---------|---------|---------|------|
```

- 金額はカンマ区切り可
- `## 設定` セクションで期間・科目マスタパスを指定
- `## 仕訳データ` セクションにテーブルを配置

### 科目マスタ

```markdown
| 科目名 | 貸借区分 | 表示順 | 決算書区分 | 説明 |
|--------|---------|--------|-----------|------|
```

- 複数テーブル対応（損益計算書科目・貸借対照表科目を別テーブルでも可）
